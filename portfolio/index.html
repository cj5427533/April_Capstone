  • 건물 클릭 시 이벤트 정보 확인 및 강의실 예약 기능
  • Express API 연동 및 예약 시스템 DB 구축

Three.js HTML/CSS JavaScript Express.js MySQL

  • 디자인툴로 UI 스케치 후 구현 → 사용자 중심 인터페이스 설계
